# RichText

> Android平台下的富文本显示TextView

* 支持Html格式文本
* 支持图片点击事件
* 链接自动回调
* 支持设置加载中和加载错误时的图片

### gradle中引用的方法

```
compile 'zhou.widget:richtext:1.0.1'
```
